<h2>🛠️ Installation Steps:</h2>

<p>1. Install dependencies</p>

```console
npm i
```

<p>2. Create .env file</p>

```console
touch .env
```

<p>3. Enter MySQL database information inside .env using following boilerplate.</p>

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
DB=
```
